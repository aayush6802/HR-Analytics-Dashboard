# HR-Analytics-Dashboard

An interactive HR Analytics Dashboard built using Power BI to analyze employee attrition trends, workforce distribution, and key HR metrics.

---

## 📊 Project Overview

This project focuses on analyzing HR data to identify employee attrition patterns across departments, job roles, salary slabs, and age groups. The dashboard helps in understanding workforce trends and supports data-driven HR decision-making.

---

## 🚀 Tools & Technologies Used

- Power BI
- Power Query
- SQL
- Excel

---

## 📌 Key Features

- Employee Attrition Analysis
- Department-wise Attrition Tracking
- Salary Slab Analysis
- Job Role & Age Group Insights
- KPI Cards & Interactive Visuals
- Dynamic Filters & Slicers

---

## 📈 Dashboard Insights

- Identified departments with higher attrition rates
- Analyzed employee distribution across salary ranges
- Tracked workforce trends using interactive KPI metrics
- Visualized attrition patterns based on age groups and job roles
